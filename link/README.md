# リンク集

## 物理
- [https://member.ipmu.jp/yuji.tachikawa/lectures/](https://member.ipmu.jp/yuji.tachikawa/lectures/): 立川先生の講義録。よくまとまっているので読んでみたい。
- [Introduction to 2d Conformal Field Theories](https://sites.google.com/site/snawata/teaching/fall-2018): 共形場理論のに関する授業。

## 数学
- [https://terrytao.files.wordpress.com/2011/01/measure-book1.pdf](https://terrytao.files.wordpress.com/2011/01/measure-book1.pdf): テレンス・タオが書いた測度論の本。一発目の演習問題から難しかった。読み応えは相当ある。

## 機械学習
- [http://banditalgs.com](http://banditalgs.com): Bandit Algorithmについてまとめた本。Banditアルゴリズムは興味あるから読みたい。有名な人が書いてるらしい。
